#실전예제: 웹의 데이터로 그래프 그리기

파이썬은 다양한 분야에서 사용하지만 요즘은 데이터 처리와 분석에 
유용하게 사용하고 있음.
웹의 데이터를 가져온 뒤 그래프로 그리는 방법을 알아보자.


1. 크롤링

웹 페이지의 HTML을 가져오기
필요한 데이터만 가져와서 파일로 저장

2.전처리

그래프로 그릴수 있도록 데이터를 가공한다.


3. 시각화

데이터로 그래프 그리기.

웹 페이지를 가져오고 파일로 저장하는 크롤링과정
데이터를 가공하는 전처리 과정
데이터로 그래프를 그리는 시각화 과정으로 구분할 수 있음.

#아나콘다 설치하기

실무에서는 시간과 노력을 절약하기 위해 패키지를 주로 이용함.
운영체제 중에서 리눅스나 macOS에서는 pip로 패키지를 설치하는데 큰 문제가없음.
윈도우에서는 pip로 패키지를 설치하더라도 중간에 에러가 나는 경우가 많음
윈도우에서는 주요패키지가 포함된 파이썬 배포판을 사용함.
파이썬 배포판은 데이터 처리 및 분석에 필요한 패키지가 모두 들어 있어서
데이터 과학 분야에서 널리 사용되고 있음.
