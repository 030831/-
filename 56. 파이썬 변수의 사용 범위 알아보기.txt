#클로저 사용하기


#변수의 사용 범위 알아보기

x=10 #전역변수
def f():
	print(x)
f()
print(x)

#위 코드 전체가 전역 범위임

실행결과
10
10

이렇게 함수를 포함하여 스크립트 전체에서 접근할수있는 변수를 전역변수 라고함.
전역 변수에 접근 할 수 있는 범위를 전역범위라고 함


def f():
	x=10 #x는 지역변수
	print(x)


f()
print(x) <- x값이 존재하지 않는다고 판단함

이때 오류가 생김

지역변수는 그 변수를 만든 함수에서만 접근할수있음.

def f():
	x=10 
	print(x)
#지역범위


x=10
def f():
	x=20
	print(x)
f()
print(x) 입력시

20
10이 출력됨
변수 이름만 같을뿐 서로 다른 변수임!

#함수 안에서 전역 변수 변경하기

x=10
def f():
	global x
	x=20
	print(x)
f()
print(x)

입력시
20
20
이 출력이 됨.

global x는 전역변수 x를 사용하겠다고 설정함.


def f():
	global x
	x=20
	print(x)
f()
print(x)
이때
20
20이 출력이 됨.
