#함수 사용하기

특정 용도의 코드를 한 곳에 모아 놓는 것을 뜻함

함수를 사용하면 코드의 용도 구분 가능
코드 재사용 가능
실수의 횟수를 줄일수있음

함수는 def (define)에 함수 이름을 지정하고 원하는 코드를 작성함
이때 코드는 반드시 들여쓰기를 해야함

ex) def 함수이름():
	코드

#Hello World! 출력함수 만들기

def hello():
	print('Hello , World!')

이제 함수 이름을 호출하면됨
hello() 엔터

단 함수를 호출하고 생성할수는없음.
